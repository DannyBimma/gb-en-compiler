Programme Description
=====================

This programme consists of 2 functions.

Function: factorial
-------------------
This function accepts one parameter named 'n' of type int, and returns a value of type int.

The function performs the following steps:

  1. If the condition "'n' is less than or equal to the value 1" is true, then:
    Return the value 1.
    
  
  2. Return the product of 'n' and call the 'factorial' function with arguments the difference between 'n' and the value 1.
  

Function: main
--------------
This function accepts no parameters and returns a value of type int.

This is the main entry point of the programme.

The function performs the following steps:

  1. Declare a variable named 'result' of type int, initialised to call the 'factorial' function with arguments the value 5.
  
  2. Return the value 0.
