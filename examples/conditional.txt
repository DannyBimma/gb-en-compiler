Programme Description
=====================

This programme consists of 2 functions.

Function: check_age
-------------------
This function accepts one parameter named 'age' of type int, and returns a value of type void.

The function performs the following steps:

  1. If the condition "'age' is greater than or equal to the value 18" is true, then:
    Display the message "You are an adult.\n".
    
    Otherwise:
    If the condition "'age' is greater than or equal to the value 13" is true, then:
      Display the message "You are a teenager.\n".
      
      Otherwise:
      Display the message "You are a child.\n".
      
    
  

Function: main
--------------
This function accepts no parameters and returns a value of type int.

This is the main entry point of the programme.

The function performs the following steps:

  1. Declare a variable named 'my_age' of type int, initialised to the value 25.
  
  2. call the 'check_age' function with arguments 'my_age'.
  
  3. Return the value 0.
